# SnakeGame

## Opis gry

SnakeGame to prosta gra konsolowa napisana w języku C#. Gracz steruje wężem za pomocą klawiszy strzałek i zdobywa punkty poprzez zbieranie obiektów pojawiających się na planszy. Gra kończy się w momencie zderzenia węża ze ścianą lub własnym ciałem.

## Instrukcja uruchomienia

1. Sklonuj repozytorium na komputer.
2. Otwórz projekt w programie Visual Studio 2022.
3. Skompiluj projekt.
4. Uruchom aplikację przy pomocy przycisku Start lub klawisza F5.
5. Steruj wężem za pomocą klawiszy strzałek.
